## 核心方法对照表

| 功能 | 方法 |
|------|------|
| 打开企业微信 | `zbbAutomation.openWechat()` |
| 文字识别 | `zbbAutomation.getAllTextNodes()` / ML Kit |
| 点击 | `zbbAutomation.tap(x, y)` |
| 长按 | `zbbAutomation.longPress(x, y, duration)` |
| 上滑 | `zbbAutomation.swipe(startX, startY, endX, endY, duration)` |
| 截图 | `zbbAutomation.screenshotViaMediaStore()` |
| 剪贴板 | `zbbAutomation.setClipboardText()` / `getClipboardText()` |
| 震动 | `zbbAutomation.startPulseVibration()` / `stopVibration()` |
| 返回键 | `zbbAutomation.pressBack()` |
| 退出应用 | 多任务键 `(300, 2300)` → 垃圾箱 `(540, 2150)` → Home `(540, 2300)` |
