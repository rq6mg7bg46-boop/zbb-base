## 一、项目概述

ZBB（自动化报备）是一款基于 Android 无障碍服务的自动化工具，用于帮助用户自动完成客户信息获取 → 微信小程序报备 → 结果反馈的完整闭环流程。

### 1.1 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Expo + React Native | 移动端 UI |
| 通信层 | React Native Bridge | JS 与 Android 通信 |
| 服务层 | AccessibilityService | Android 无障碍服务 |
| 截图 | MediaProjection API | 屏幕截图 |
| OCR | Google ML Kit | 文字识别 |

### 1.2 APP 包名

```typescript
const APP_PACKAGES = {
  DOUYIN: 'com.ss.android.ugc.aweme',  // 抖音
  WECHAT: 'com.tencent.mm',            // 微信
};
```

## 二、抖音端流程（获取客户信息）

### 2.1 流程概述

| 阶段 | 说明 |
|------|------|
| 步骤1 | 打开抖音 |
| 步骤2 | 点击"消息"按钮 |
| 步骤3 | 查找好友"只如初见"并进入聊天 |
| 步骤4 | 验证已进入对话框 |
| 步骤5 | 长按消息触发复制菜单 |
| 步骤6 | 点击"复制"按钮 |
| 步骤7 | 读取剪贴板（客户信息） |

### 2.2 详细步骤

| 步骤 | 操作 | 实现逻辑 | 坐标/方法 |
|------|------|----------|-----------|
| 1 | 打开抖音 | `launchApp(APP_PACKAGES.DOUYIN)` | - |
| 2 | 点击"消息"按钮 | 固定坐标点击 | `(750, 2300)` |
| 3 | 查找好友"只如初见" | 动态查找并点击 | 节点树解析 |
| 4 | 进入聊天 | 验证进入对话框 | 检查"只如初见"或"发送消息" |
| 5 | 长按消息 | OCR识别消息位置，长按2秒 | 消息区域动态坐标 |
| 6 | 点击"复制"按钮 | 查找"复制"节点并点击 | 动态坐标/兜底`(270, 1350)` |
| 7 | 读取剪贴板 | 读取并解析客户信息 | - |

### 2.3 核心方法

```typescript
// 打开抖音
await zbbAutomation.launchApp(APP_PACKAGES.DOUYIN);

// 长按消息
await zbbAutomation.longPress(x, y, 2000);

// 点击复制
await zbbAutomation.click(x, y);

// 读取剪贴板
const clipboardText = await zbbAutomation.getClipboardText();
```

### 3.1 流程概述

### 3.2 详细步骤

| 步骤 | 操作 | 实现逻辑 | 说明 |

| 3 | 点击"工作台" | `findElementByText('工作台')` | 动态查找 |
| 4 | 点击"越秀地产悦秀会" | `findElementByText('越秀地产悦秀会')` | 动态查找 |
| 5 | 点击"我要推荐" | `findTextByMLKit('我要推荐')` | ML Kit 识别 |
| 6 | 等待页面加载 | `delay(5000)` | - |

## 五、核心方法对照表

| 功能 | 方法 | 说明 |
|------|------|------|
| 打开应用 | `launchApp(packageName)` | 启动指定应用 |
| 点击 | `click(x, y)` / `tap(x, y)` | 点击指定坐标 |
| 长按 | `longPress(x, y, duration)` | 长按指定坐标 |
| 查找文字 | `findElementByText(text)` | 使用无障碍服务查找 |
| 文字识别 | `findTextByMLKit(text)` | 使用 ML Kit OCR |
| 截图 | `screenshotViaMediaStore()` | 截图保存到相册 |
| 剪贴板 | `getClipboardText()` / `setClipboardText()` | 读写剪贴板 |
| 震动 | `startPulseVibration()` / `stopVibration()` | 脉冲震动 |
| 延迟 | `delay(ms)` | 等待指定毫秒 |
| 返回键 | `pressBack()` | 点击系统返回键 |
| 退出应用 | 多任务键→垃圾箱→Home键 | 三键组合退出 |
